# Contributing to TradingXbert

Thank you for your interest in contributing to TradingXbert!

## Initial Setup

If you're setting up this repository for the first time on your local machine, follow these steps:

### For Windows Users

```cmd
cd C:\Users\YourUsername\TradingXbert
git remote add origin https://github.com/YOUR-USERNAME/tradingxbert.git
git branch -M main
git push -u origin main
```

### For macOS/Linux Users

```bash
cd ~/TradingXbert
git remote add origin https://github.com/YOUR-USERNAME/tradingxbert.git
git branch -M main
git push -u origin main
```

**Note:** Replace `YOUR-USERNAME` with your actual GitHub username.

## Development Workflow

1. Fork the repository
2. Clone your fork locally
3. Create a new branch for your feature or bugfix
4. Make your changes
5. Commit your changes with a descriptive commit message
6. Push to your fork
7. Submit a pull request

## Code Style

Please follow the existing code style and conventions used in the project.

## Questions?

If you have any questions, feel free to open an issue in the repository.
